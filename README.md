# Double Time Games — Website

A clean, minimal website for Double Time Games, an indie board game publisher.

## Brand Principles

- **Rhythm over noise**: Paced, intentional, controlled
- **Confidence through restraint**: Space as a feature
- **System over novelty**: Repeatable framework
- **Longevity first**: Timeless, never trendy

## Color Palette

- **Primary Dark**: `#0F0F12` (near-black)
- **Primary Light**: `#F4F2EE` (warm off-white)
- **Accent**: `#1ED4C6` (electric teal)

## Typography

- **Font**: Helvetica Neue
- **Weights**: SemiBold (titles), Medium (headers), Regular (body)

## File Structure

```
.
├── index.html          # Main website page
├── styles.css          # All styling
├── assets/
│   ├── logo-primary.svg    # Wordmark logo
│   ├── rhythm-mark.svg     # || symbol
│   └── logo-combined.svg   # Wordmark + rhythm mark
└── README.md           # This file
```

## Running Locally

Simply open `index.html` in a web browser, or use a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Design Notes

- Generous white space throughout
- Left-aligned layouts
- Minimal decoration
- Accent color used sparingly for dividers and rhythm mark
- Mobile responsive design
